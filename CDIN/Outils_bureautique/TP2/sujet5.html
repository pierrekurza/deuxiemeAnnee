TP 1 : prise en main de Libre Office

Ce premier TP est consacré à la prise en mains du logiciel libre de traitement de texte LibreOffice Writer, qui est compatible en lecture et en écriture avec les fichiers issus de Microsoft Word, le logiciel le plus utilisé en bureautique pour le traitement de texte (qui n'est, lui, ni libre ni gratuit).

Lors de ce premier TP, vous allez commencer à mettre en forme à l’aide du logiciel de traitement de texte Writer un texte brut selon un modèle.

Exercice 1 : ouvrir et sauvegarder les fichiers au bon format

  1. Récupérez le texte brut ainsi que le résultat attendu sous Moodle.
  2. Lancez Libre Office Writer.
  3. Ouvrez le fichier récupéré.
  4. Sauvegardez maintenant votre document avec Enregistrer sous. Dans la fenêtre qui apparaît, pensez à bien spécifier :
    • le nom du fichier,
    • le répertoire dans lequel il va être enregistré,
    • le format sous lequel il va être enregistré.

Après la fermeture de la fenêtre, sur la barre supérieure de la fenêtre LibreOffice Writer, le nom du fichier apparaît.

Si vous souhaitez avoir la possibilité de lire ultérieurement votre fichier avec Microsoft Word, vous devez choisir comme Type de fichier un des formats Microsoft Word (.doc*).

Pensez à enregistrer régulièrement vos modifications. En cas de fausse manœuvre, vous pourrez récupérer le travail accompli jusqu'au dernier enregistrement.

Exercice 2 : mettre le texte en forme

Vous devez maintenant mettre en forme votre document selon le modèle fourni. Pour vous aider dans la mise en forme du texte, vous pouvez consulter le document de cours fourni dans Moodle.

  1. Affichez les caractères non imprimables en cliquant sur l'icône de paragraphe dans la barre d'outils de Writer (ou dans le menu Affichage).
  2. Appliquez la mise en forme du texte.
  3. Vérifiez ensuite les paramètres de la mise en page du document.
  4. Vérifiez le résultat à l’aide de la commande Aperçu.

Exercice 3 : rendre le devoir sous Moodle

Quand vous avez terminé votre travail, déposez votre fichier au format LibreOffice enregistré sous le nom tp1NomPrenomGroupe.odt ) dans le devoir créé sous Moodle à cet effet.
